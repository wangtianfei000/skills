# Remotion 类PPT视频模板

本文件只保存三个模板：预览 HTML、Remotion 工程模板、Remotion 代码模板。不要在这里放小程序展示用 JSON、分镜数据结构或草稿数据结构。

## 预览 HTML 模板

function fitStage(){
  var stage = document.getElementById('stage');

function renderDots(){
  dotsRoot.innerHTML = '';
  slides.forEach(function(_, index){
    var dot = document.createElement('span');
    dot.className = 'dot' + (index === currentIndex ? ' active' : '');

  currentIndex = Math.max(0, Math.min(slides.length - 1, index));
  slides.forEach(function(slide, i){ slide.className = slide.className.replace(' active','') + (i === currentIndex ? ' active' : ''); });

## Remotion 工程模板

用途：用户确认后，直接复制本 skill 自带的 Remotion 工程模板，作为视频生成工程的起点。

```text
assets/remotion-doc-renderer-template/
  package.json
  package-lock.json
  tsconfig.json
  tsconfig.build.json
  README.md
  tools/
    validate-scene.js
  src/
    index.ts
    Root.tsx
    render.ts
    doctor.ts
    browser-download.ts
    browser-check.ts
    generated/
      scene.generated.tsx
    compositions/
      GeneratedDocVideo.tsx
    blocks/
      index.tsx
      HeroBlock.tsx
      CardGridBlock.tsx
      ComparisonBlock.tsx
      WorkflowBlock.tsx
      MetricCardsBlock.tsx
      CodeChatBlock.tsx
      TableBlock.tsx
      CtaBlock.tsx
      shared.tsx
```

使用方式：

- 将 `assets/remotion-doc-renderer-template/` 复制到目标任务目录或目标项目目录。
- 进入复制后的工程目录安装依赖。
- 使用确认后的分镜内容生成 `scene.generated.tsx`。
- 先确定配音方案，再生成分段旁白音频；音频建议放在任务目录的 `audio/` 下，例如 `audio/segment_001.mp3`、`audio/segment_002.mp3`。
- 为每段音频记录真实时长，并用真实时长换算每页 `from` 和 `durationInFrames`。
- 如果采用模板的 manifest 渲染入口，把 `scene.generated.tsx`、音频段落、BGM、输出路径写入任务目录，并运行 `node dist/render.js --manifest <manifest.json>`。
- 如果采用 Remotion Studio/CLI，可直接改 `src/generated/scene.generated.tsx` 并运行工程脚本。
- 保留 `src/blocks/` 中的通用页面 block，可作为兜底组件；更推荐根据确认版预览 HTML 直接写 Remotion JSX/CSS。
- 不要把 `node_modules`、`dist`、临时输出视频复制进 skill 资源或最终模板。

模板中的关键入口：

- `src/render.ts`：Node 端程序化渲染入口，负责 bundle、selectComposition、renderMedia、renderStill。
- `src/generated/scene.generated.tsx`：默认生成场景文件，实际制作时用确认后的 Remotion 代码替换。
- `src/compositions/GeneratedDocVideo.tsx`：组合生成场景。
- `src/Root.tsx` 和 `src/index.ts`：Remotion Composition 注册入口。
- `tools/validate-scene.js`：可用于检查受限 scene 代码和时间轴。
- `src/browser-download.ts`、`src/browser-check.ts`：部署或本地环境中处理浏览器依赖。

音频任务目录建议：

```text
task/
  manifest.json
  scene.generated.tsx
  audio/
    segment_001.mp3
    segment_002.mp3
    bgm.mp3
  final.mp4
  cover.jpg
```

配音记录建议：

```text
配音方案：普通话，稳重讲解型男声/女声
音色 ID：zh-CN-YunjianNeural / zh-CN-XiaoxiaoNeural / zh-CN-XiaoyiNeural / zh-CN-YunxiNeural
语速：自然，不抢节奏
情绪：专业、清晰、可信
TTS 输入：
- 第 1 页：使用确认版第 1 页旁白，输出 audio/segment_001.mp3
- 第 2 页：使用确认版第 2 页旁白，输出 audio/segment_002.mp3
音频：
- 第 1 页：audio/segment_001.mp3，真实时长 6.2 秒，视觉时长 6.8 秒
- 第 2 页：audio/segment_002.mp3，真实时长 7.4 秒，视觉时长 8.0 秒
BGM：可选，低音量铺底；有人声时降低音量
```

项目实现参考：

- 使用 skill 自带的 `scripts/tts_util.py` 和 `scripts/tts_voices.py` 生成 EdgeTTS MP3；不要依赖原项目包路径。
- 批量生成分段音频时，优先运行 `scripts/generate_edge_tts_segments.py`。
- 音色使用 EdgeTTS voice id，例如 `zh-CN-YunjianNeural`。
- 按页面逐段调用 TTS，不要合成整片长音频。
- Remotion 文档视频链路先生成音频，再生成 `manifest.json` 和视频代码。
- 每个音频文件生成后，用 `ffmpeg.probe()` 读取真实时长。
- 音频帧数按 `ceil(真实秒数 * fps) + 12` 计算，12 帧作为尾部缓冲。
- `manifest.json` 的 `audio.segments` 由真实音频信息组成，Remotion 渲染入口会把这些段落映射为 `<Audio>`。
- 如果需要 BGM，可生成 `audio/bgm.mp3`，音量保持很低；有人声时进一步降低。

分段音频输入示例：

```json
{
  "voice": "zh-CN-YunjianNeural",
  "speed": 1.35,
  "fps": 30,
  "targetDurationSeconds": 40,
  "scenes": [
    {"id": "scene_001", "narration": "这里放第 1 页确认后的旁白。"},
    {"id": "scene_002", "narration": "这里放第 2 页确认后的旁白。"}
  ]
}
```

运行示例：

```bash
python scripts/generate_edge_tts_segments.py --input task/narration.json --output-dir task
```

## Remotion 代码模板

用途：把确认后的分镜预览转换为真实视频画面。按分镜页数复制页面组件和 `Sequence`，并根据旁白时长调整 `from` 与 `durationInFrames`。

```tsx
import React from "react";
import {
  AbsoluteFill,
  Audio,
  Sequence,
  interpolate,
  spring,
  staticFile,
  useCurrentFrame,
  useVideoConfig,
} from "remotion";

const width = 1920;
const height = 1440;
const fps = 30;
const clamp = {extrapolateLeft: "clamp" as const, extrapolateRight: "clamp" as const};

const narration = [
  {
    id: "scene_001",
    src: "audio/segment_001.mp3",
    from: 0,
    durationSeconds: 6.2,
    audioDurationInFrames: 198,
    visualDurationInFrames: 210,
  },
  {
    id: "scene_002",
    src: "audio/segment_002.mp3",
    from: 210,
    durationSeconds: 7.4,
    audioDurationInFrames: 234,
    visualDurationInFrames: 240,
  },
];

export const videoMeta = {
  width,
  height,
  fps,
  durationInFrames: 450,
};

const pageStyle: React.CSSProperties = {
  background:
    "radial-gradient(circle at 50% 20%, rgba(105,242,213,0.16), transparent 34%), linear-gradient(135deg, #070B1A, #111827)",
  fontFamily: "\"Noto Sans SC\",\"PingFang SC\",\"Microsoft YaHei\",sans-serif",
  color: "#fff",
  display: "flex",
  alignItems: "center",
  justifyContent: "center",
};

const safeStyle: React.CSSProperties = {
  width: "86%",
  height: "82%",
  display: "flex",
  flexDirection: "column",
  alignItems: "center",
  justifyContent: "center",
  textAlign: "center",
  gap: 36,
};

const SceneOne: React.FC = () => {
  const frame = useCurrentFrame();
  const {fps} = useVideoConfig();
  const opacity = interpolate(frame, [0, 24], [0, 1], clamp);
  const y = interpolate(frame, [0, 24], [28, 0], clamp);
  const pulse = spring({frame, fps, config: {damping: 18, stiffness: 90}});

  return (
    <AbsoluteFill style={pageStyle}>
      <div style={safeStyle}>
        <div style={{fontSize: 34, color: "#69F2D5", fontWeight: 700}}>短标签</div>
        <h1
          style={{
            fontSize: 92,
            lineHeight: 1.1,
            margin: 0,
            color: "#69F2D5",
            opacity,
            transform: `translateY(${y}px) scale(${0.98 + pulse * 0.02})`,
          }}
        >
          中文标题
        </h1>
        <p style={{fontSize: 46, lineHeight: 1.35, maxWidth: "80%", color: "#D7E0EF", margin: 0}}>
          中文副标题或核心说明。
        </p>
      </div>
    </AbsoluteFill>
  );
};

const SceneTwo: React.FC = () => {
  const frame = useCurrentFrame();
  const items = ["亮点一", "亮点二", "亮点三"];
  return (
    <AbsoluteFill style={pageStyle}>
      <div style={safeStyle}>
        <h2 style={{fontSize: 76, color: "#69F2D5", margin: 0}}>核心亮点</h2>
        <div style={{display: "grid", gridTemplateColumns: "repeat(3, 1fr)", gap: 28, width: "100%"}}>
          {items.map((item, index) => {
            const delay = 12 + index * 8;
            const opacity = interpolate(frame, [delay, delay + 24], [0, 1], clamp);
            const y = interpolate(frame, [delay, delay + 24], [24, 0], clamp);
            return (
              <div
                key={item}
                style={{
                  minHeight: 220,
                  border: "2px solid rgba(105,242,213,0.42)",
                  borderRadius: 24,
                  background: "rgba(17,24,39,0.86)",
                  display: "flex",
                  alignItems: "center",
                  justifyContent: "center",
                  textAlign: "center",
                  fontSize: 40,
                  fontWeight: 800,
                  opacity,
                  transform: `translateY(${y}px)`,
                }}
              >
                {item}
              </div>
            );
          })}
        </div>
      </div>
    </AbsoluteFill>
  );
};

export const Video: React.FC = () => {
  return (
    <AbsoluteFill>
      <Sequence from={0} durationInFrames={210}>
        <SceneOne />
      </Sequence>
      <Sequence from={210} durationInFrames={240}>
        <SceneTwo />
      </Sequence>
      {narration.map((item) => (
        <Sequence key={item.id} from={item.from} durationInFrames={item.audioDurationInFrames}>
          <Audio src={staticFile(item.src)} />
        </Sequence>
      ))}
      <Audio
        src={staticFile("audio/bgm.mp3")}
        volume={(frame) => {
          const hasVoice = narration.some((item) => frame >= item.from && frame < item.from + item.audioDurationInFrames);
          return hasVoice ? 0.035 : 0.08;
        }}
      />
    </AbsoluteFill>
  );
};
```
